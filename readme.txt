Registro público de promesas mundialistas con tabla propia y páginas:
- /registro/
- /promesas-realizadas/
- /promesa/{hash}/

== Instalación ==
1. Subir el plugin y activarlo.
2. Crear páginas:
   - Registro (slug: registro)
   - Promesas realizadas (slug: promesas-realizadas)
3. Ir a Ajustes > Enlaces permanentes y Guardar cambios.

== Notas ==
- No incluye captcha ni medidas anti-bot. (Se gestiona fuera del plugin).
- Buscador: solo por Nombre y Apellido.
- Imágenes: no incluidas en esta versión.
NOTAS:
Debe ser acompañado por el plugin de censura de malas palabras censura.php que s epuede cargar dentro de la carpeta mu-plugin de wordpress
